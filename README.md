## Gitty: self hosted Git manamgement sofeware ##
-------------------------------------------------

### Gitty 能提供 ###
*	保证服务器上的代码安全
*	快速浏览项目的files,commits,branchs,tags
*	共享团队项目

### Gitty 基于 ###
*	CodeIgniter 一个用 PHP 编写的应用程序开发框架.
*	开放源代码

### 配置 ###
*	Ubuntu/Debian**
*	PHP 版本5.1.6或更新
*	MySQL(4.1+)
*	git
*	openssh
*	gitosis
